# 神兽转向牌

一个纯前端网页版单机牌局。真人玩家对战 3 个 AI，对局比分会在浏览器本地持续累计。

## 运行

直接打开 `index.html` 即可游玩；也可以在项目目录启动一个静态服务：

```sh
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 已实现规则

- 55 张牌：36 张数字牌、4 张大王、6 张小王、9 张神兽万能牌。
- 每名玩家开局 5 张手牌，剩余为抽牌堆，翻 1 张作为弃牌堆顶牌。
- 开局翻到大王或神兽万能牌会收回重翻；开局翻到小王会反转方向并改由庄家右手侧先手。
- 数字牌按同花色或同数字匹配；大王和神兽可任意打出并指定生效花色。
- 小王按花色匹配，或在万能牌之后接续打出，打出后立即反转方向。
- 无合规手牌时必须摸 1 张；刚摸到的牌若能打出，可以立刻打出。
- 率先打空手牌者获得 1 分，比分保存在浏览器本地存储中并跨局累计。
- 牌面 UI 使用扑克牌样式：数字牌显示 1-9 点数，大王/小王显示为带图案的 Joker 牌面。
- AI 会先进入思考状态再行动，出牌时有从座位飞向弃牌堆的动画和来源标签。
- 万能牌和小王会在独立区域展示效果，方向变化会用顺/逆时针图标和醒目提示呈现。
- 青龙、白虎、朱雀使用各自独立的神兽图案卡面。

为了让“小王需同花色匹配”的规则可执行，6 张小王实现为带花色小王，分布为黑桃 2、红桃 2、梅花 1、方块 1。
